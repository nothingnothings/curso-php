--> COMO É UMA SCRIPTING LANGUAGE,



É UMA PROGRAMMING LANGUAGE...






a diferenca principal entre 

SCRIPTING LANGUAGES E PROGRAMMING LANGUAGES 



RESIDE NA SUA EXECUÇAO....






ENQUANTO QUE 

PROGRAMMING LANGUAGES PRECISAM SER COMPILADAS

EM __ MACHINE LANGUAGE PARA SEREM EXECUTADAS,





AS SCRIPTING LANGUAGES PRECISAM 



SER __ INTERPRETADAS__ (interpreted)....










EXEMPLO:







PROGRAMMING LANGUAGE --> JAVA, C OU C++ 


SOURCE CODE --> MACHINE CODE --> MACHINE...









SCRIPTING LANGUAGE:



SOURCE CODE --> INTERPRETER....








--> java, c, c++ --> sao programming LANGUAGES 

que 

precisam ser compiladas 


para serem executadas...









--> JAVASCRIPT E PHP ---> 


ESSAS 2 PROGRAMMING/SCRIPTING LANGUAGES 


APENAS PRECISAM DE 1 __ INTERPRETADOR__ 

PARA SEREM EXECUTADAS....









--> AO CONTRÁRIO DO JAVASCRIPT,


QUE RODA __ NO CLIENT (bROWSEr),


O PHP 



PRECISA DE 1 SERVER PARA PODER RODAR...








--> UMA REPRESENTACAO BÁSICA DE COMO O PHP FUNCIONA:









CLIENT (BROWSER) ---> DISPARA REQUEST 


CONTRA O SERVER 



SERVER --> TEM O INTERPRETER,

INTERPRETA E PROCESSA PHP --->  






O SERVER TAMBEM PODE FAZER 


OUTRAS COISAS NO SEU INTERIOR,

COMO SE CONECTAR A DATABASES 

E CONVERSAR COM APIs...









FINALMENTE,



O SERVER __ RESPONDE,


ENVIA A RESPONSE 


AO BROWSER...
